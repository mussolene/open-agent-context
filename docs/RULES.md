# Rules / Правила

## EN
Rules can block, warn, require confirmation, annotate context, or rank only.
Built-ins: no secret leak, no unconfirmed sensitive memory commit, fuzzy memory
not fact, deny tool write without capability, audit all memory access.

## RU
Правила могут block, warn, require confirmation, annotate context или rank only.
Встроены: no secret leak, no unconfirmed sensitive memory commit, fuzzy memory
not fact, deny tool write without capability, audit all memory access.
