# MCP Bindings / MCP Bindings

## EN
MCP bindings are typed adapter definitions. The POC imports MCP config JSON,
stores server/tool metadata, and exposes imported tools through the tool
registry. It does not run a full MCP client.

## RU
MCP bindings — типизированные adapter definitions. POC импортирует MCP config
JSON, хранит metadata server/tool и показывает импортированные tools через
реестр tools. Полный MCP client не запускается.
