# Benchmark / Benchmark

## EN
The benchmark compares `baseline_no_memory` with `oacs_memory_loop`.
Synthetic tasks are local and deterministic. Scoring checks required facts,
forbidden facts, superseded memory handling, clarification behavior, JSON
validity, latency, token estimates, memory used, and audit events.
